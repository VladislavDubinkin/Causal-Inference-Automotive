# Причинный вывод на примере немецких автомобильных акций
### Оценка влияния дизельного скандала (Dieselgate) и COVID-19 на волатильность акций
### с использованием CausalImpact (BSTS) и синтетического контроля

## Бизнес-задача:

Специалистам по риск-менеджменту необходимо различать **специфические для компании шоки** (например, корпоративный скандал)
от **системных шоков** (например, пандемия) — для каждого типа требуются разные стратегии хеджирования.
В данном проекте количественно оцениваются оба типа шоков с помощью методовного вывода на основе 16-летних
ежедневных данных по акциям Volkswagen, BMW и Mercedes-Benz.

---

## Гипотезы:

| № | Гипотеза |
|---|-----------|
| H₁: | Дизельный скандал вызвал **статистически значимый** всплеск волатильности VW, но **не** у BMW/MB |
| H₂: | COVID-19 вызвал **синхронизированный** шок у всех трёх производителей |
| H₃: | Масштаб эффекта COVID-19 превышает эффект дизельного скандала для всех компаний |

---

## Методы:

| Событие | Метод | Почему |
|-------|--------|-----|
| Дизельный скандал (сентябрь 2015) | **CausalImpact** (BSTS) | Короткий постпериод, доступно множество контрольных единиц |
| COVID-19 (февраль 2020) | **Синтетический контроль** | Длительный устойчивый шок, академический стандарт |
| Валидация | **Плацебо-тесты с перестановкой** | Непараметрическое p-значение для обоих методов |

---

## Значение для риск-менеджмента:

- **Стресс-тестирование:** шок волатильности, откалиброванный по дизельному скандалу (+18 п.п.),
  может служить сценарием для портфелей автомобильного сектора
- **Корректировка VaR:** масштаб шока COVID-19 указывает, что стандартный исторический VaR
  недооценивает хвостовой риск во время системных событий примерно в 2 раза
- **Хеджирование:** различие между специфическим для компании и системным шоком меняет
  оптимальный коэффициент хеджирования — отраслевой ETF хеджирует COVID-19, но не дизельный скандал
